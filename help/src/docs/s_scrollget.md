# S.scrollGet

## SINTAXIS
```javascript
S.scrollGet(obj)
```

## DESCRIPCIÓN
Obtiene la posición de scroll de un elemento.

## PARÁMETROS
- `obj` (Element): Elemento del cual obtener la posición de scroll

## EJEMPLO
```javascript
var scrollPos = S.scrollGet(document.body);
console.log('Scroll position:', scrollPos);
```
