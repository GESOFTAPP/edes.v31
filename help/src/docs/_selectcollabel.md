# .selectColLabel

## SINTAXIS
```javascript
S().selectColLabel(n)
```

## DESCRIPCION
Selecciona la etiqueta de una columna específica en un listado o tabla.

## PARAMETROS
- `n`: Número de la columna a seleccionar (empezando desde 0)

## EJEMPLO
```javascript
S().selectColLabel(2); // Selecciona la etiqueta de la tercera columna
```
