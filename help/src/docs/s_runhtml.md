# S.runHTML

## SINTAXIS
```javascript
S.runHTML(win, html)
```

## DESCRIPCIÓN
Ejecuta código HTML en la ventana especificada.

## PARÁMETROS
- `win` (Window): Ventana donde ejecutar el HTML
- `html` (string): Código HTML a ejecutar

## EJEMPLO
```javascript
S.runHTML(window, '<div>Contenido dinámico</div>');
```
