# S.modal

## SINTAXIS
```javascript
S.modal(css)
```

## DESCRIPCIÓN
Crea y muestra un modal (ventana emergente) con los estilos CSS especificados.

## PARÁMETROS
- `css` (string): Estilos CSS para aplicar al modal

## EJEMPLO
```javascript
S.modal('width: 400px; height: 300px; background: white;');
```
