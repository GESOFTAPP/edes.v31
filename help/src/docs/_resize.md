# .resize

## SINTAXIS
```javascript
S().resize(minw, minh)
```

## DESCRIPCION
Redimensiona un elemento estableciendo un tamaño mínimo.

## PARAMETROS
- `minw`: Ancho mínimo en píxeles
- `minh`: Alto mínimo en píxeles

## EJEMPLO
```javascript
S('#miElemento').resize(300, 200); // Establece tamaño mínimo 300x200
```
