# S.rtrim

## SINTAXIS
```javascript
S.rtrim(txt)
```

## DESCRIPCIÓN
Elimina espacios en blanco al final (derecha) de una cadena de texto.

## PARÁMETROS
- `txt` (string): Texto del cual eliminar espacios al final

## EJEMPLO
```javascript
S.rtrim("Hello World   ")    // "Hello World"
S.rtrim("  texto  ")         // "  texto"
S.rtrim("sin espacios")      // "sin espacios"
```
