# .goto

## SINTAXIS
```javascript
S().goto(x, y)
```

## DESCRIPCION
Posiciona un elemento en coordenadas específicas de la pantalla.

## PARAMETROS
- `x`: Posición horizontal (píxeles)
- `y`: Posición vertical (píxeles)

## EJEMPLO
```javascript
S('#miElemento').goto(100, 200); // Posiciona el elemento en x=100, y=200
```
