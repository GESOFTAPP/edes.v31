# S.hex2rgb

## SINTAXIS
```javascript
S.hex2rgb(c)
```

## DESCRIPCIÓN
Convierte un color en formato hexadecimal a formato RGB.

## PARÁMETROS
- `c` - Color en formato hexadecimal (ej: "#FF0000" or "FF0000")

## EJEMPLO
```javascript
// Convertir color hexadecimal a RGB
let rgb = S.hex2rgb("#FF0000");
// Resultado: {r: 255, g: 0, b: 0}
```
