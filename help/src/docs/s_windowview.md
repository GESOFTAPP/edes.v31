# S.windowView

## SINTAXIS
```javascript
S.windowView(o)
```

## DESCRIPCIÓN
Gestiona la vista de una ventana específica.

## PARÁMETROS
- **o**: Objeto o configuración de la vista de ventana

## EJEMPLO
```javascript
S.windowView({
    id: 'miVista',
    visible: true,
    maximized: false
});
```
