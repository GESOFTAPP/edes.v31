# S.windowHidden

## SINTAXIS
```javascript
S.windowHidden(win)
```

## DESCRIPCIÓN
Verifica si una ventana está oculta.

## PARÁMETROS
- **win**: Objeto ventana a verificar

## EJEMPLO
```javascript
let estaOculta = S.windowHidden(miVentana);
if (estaOculta) {
    console.log('La ventana está oculta');
}
```
