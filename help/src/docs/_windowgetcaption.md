# .windowGetCaption

## SINTAXIS
```javascript
S().windowGetCaption(conTag)
```

## DESCRIPCION
Obtiene el título o caption actual de una ventana.

## PARAMETROS
- `conTag`: (opcional) Incluir tags HTML en el resultado

## EJEMPLO
```javascript
var titulo = S().windowGetCaption(false); // Obtiene el título sin tags HTML
```
