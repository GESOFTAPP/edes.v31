# .menu

**SINTAXIS**
```javascript
S().menu(op, para, arg)
```

**DESCRIPCIÓN**
Crea un menú interactivo.

**PARÁMETROS**
- `op`: Opciones del menú
- `para`: Parámetros adicionales
- `arg`: Argumentos de configuración

**EJEMPLO**
```javascript
S("#menu-contenedor").menu(opciones, parametros, argumentos);
```
