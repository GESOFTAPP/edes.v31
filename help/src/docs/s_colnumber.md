# S.colNumber

## SINTAXIS
```javascript
S.colNumber(nmField)
```

## DESCRIPCIÓN
En un listado te devuelve el número de columna del nombre de un campo.

## PARÁMETROS
- **nmField**: String con el nombre del campo a buscar

## EJEMPLO
```javascript
let numeroColumna = S.colNumber("cd_auto");
console.log(numeroColumna);
```
