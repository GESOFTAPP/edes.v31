# S.progressText

## SINTAXIS
```javascript
S.progressText(txt)
```

## DESCRIPCIÓN
Actualiza el texto de una barra de progreso o indicador de progreso.

## PARÁMETROS
- `txt` (string): Texto a mostrar en el progreso

## EJEMPLO
```javascript
S.progressText("Procesando archivo 1 de 10...");
S.progressText("Guardando datos...");
```
