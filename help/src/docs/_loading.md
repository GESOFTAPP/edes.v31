# .loading

**SINTAXIS**
```javascript
S().loading(on)
```

**DESCRIPCIÓN**
Controla el estado de carga/loading.

**PARÁMETROS**
- `on`: Booleano para activar/desactivar el loading

**EJEMPLO**
```javascript
S("#contenedor").loading(true);  // Activar loading
S("#contenedor").loading(false); // Desactivar loading
```
