# .nodeMoveFirst

**SINTAXIS**
```javascript
S().nodeMoveFirst(item3)
```

**DESCRIPCIÓN**
Mueve un elemento a la primera posición del contenedor especificado.

**PARÁMETROS**
- `item3`: Elemento contenedor de destino

**EJEMPLO**
```javascript
S("#elemento-a-mover").nodeMoveFirst("#contenedor-destino");
```
