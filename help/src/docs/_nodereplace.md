# .nodeReplace

**SINTAXIS**
```javascript
S().nodeReplace(oNew)
```

**DESCRIPCIÓN**
Reemplaza el elemento seleccionado con un nuevo elemento.

**PARÁMETROS**
- `oNew`: Nuevo elemento que reemplazará al actual

**EJEMPLO**
```javascript
S("#elemento-original").nodeReplace("<div>Nuevo contenido</div>");
```
