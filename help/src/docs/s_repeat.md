# S.repeat

## SINTAXIS
```javascript
S.repeat(c, n)
```

## DESCRIPCIÓN
Repite un carácter o cadena un número específico de veces.

## PARÁMETROS
- `c` (string): Carácter o cadena a repetir
- `n` (number): Número de repeticiones

## EJEMPLO
```javascript
S.repeat("*", 5)        // "*****"
S.repeat("abc", 3)      // "abcabcabc"
S.repeat("-", 10)       // "----------"
```
