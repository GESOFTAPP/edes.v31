# .around

## SINTAXIS
```javascript
S().around(tip, p)
```

## DESCRIPCIÓN
Método para posicionar elementos alrededor de otros elementos.

## PARÁMETROS
- `tip`: Tipo de posicionamiento
- `p`: Parámetros adicionales de configuración

## EJEMPLO
```javascript
S("#elemento").around("top", {offset: 10});
```
